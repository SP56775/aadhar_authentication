# Aadhaar Authentication Count Analysis using SQL

## Project Overview

This project analyzes Aadhaar authentication count data using SQL to identify patterns and trends.

The analysis includes:

* Time-based trend analysis
* Age group comparison
* Demographic vs Biometric authentication comparison
* District-wise highest authentication
* State-wise authentication comparison
* Pincode-based analysis
* Outlier detection
* Biometric ratio analysis
* Correlation analysis

## Dataset Columns

* date
* pincode
* age_0_5
* age_5_17
* age_18_greater
* demo_age_5_17
* demo_age_17_
* district
* bio_age_5_17
* bio_age_17_
* state_standardized

## SQL Queries Used

* Total authentication by date
* Age-wise authentication totals
* Demographic and biometric comparison
* District and state-level analysis
* High authentication district detection
* Biometric ratio calculation
* Correlation between demographic and biometric authentication
